# xxx-web
欢迎来到手势粒子系统！这是一个基于 Web 的交互式艺术项目，它利用您的摄像头捕捉手部动作，实时操控一个由数万颗粒子组成的绚丽 3D 世界。在这里，您的手就是魔法棒！

## ✨ 核心功能

### 1. 实时手势追踪与互动

本系统通过 Google 的 MediaPipe Hands 技术，能够精准识别您在摄像头前的单手手势，并将其转化为对粒子系统的控制指令。

- **粒子聚散**：
    - **张开手掌** `🖐️`：粒子云会向外扩散，呈现出爆炸或星云的宏大效果。
    - **握拳或捏合** `✊`：粒子会迅速向中心聚拢，形成一个密集的光核。

- **粒子形态切换**：
    - 系统会识别您伸出的手指数量（1-4根），并命令粒子组成对应的文字。
    - **☝️ 一根手指**：显示 “Hello World” (科技蓝色调)
    - **✌️ 两根手指**：显示 “2026” (璀璨金色调)
    - **👌 三根手指**：显示 “新年快乐” (喜庆红色调)
    - **👍 四根手指**：显示 “马到成功” (活力绿色调)

### 2. 动态文字与色彩主题

粒子不仅能组成文字，还会根据当前显示的文字切换整体的色彩风格，为您带来沉浸式的视觉体验。

- 每个文字主题都拥有一套独特且和谐的配色方案。
- 组成文字的粒子会更加明亮，而背景中的粒子则会以较暗的同色系粒子作为点缀，增强了场景的层次感和氛围感。

### 3. 高度自定义内容

为了让体验更加个性化，您可以随时修改手势对应的文字！

- **如何操作**：在页面上**快速双击空格键**。
- **功能**：会弹出一个输入框，您可以按照 `文字1,文字2,文字3,文字4` 的格式，一次性修改 1-4 根手指分别对应的显示内容。
- **实时生效**：修改后，系统会立刻更新指令，当您做出相应手势时，粒子就会组成您刚刚输入的新文字。

### 4. 沉浸式 3D 视觉效果

项目基于强大的 `Three.js` 库构建，提供了流畅且华丽的 3D 视觉效果。

- **自动旋转**：镜头会围绕粒子中心缓慢旋转，让您从不同角度欣赏粒子形态的美感。
- **自由视角**：您可以使用鼠标拖动、滚轮缩放，自由地探索这个粒子宇宙的每一个角落。
- **平滑过渡**：无论是粒子的位置变化还是颜色切换，都采用了平滑的插值动画，使得所有状态间的过渡都如丝般顺滑，毫无顿挫感。

## 🚀 技术栈

- **3D 渲染引擎**: [Three.js](https://threejs.org/)
- **手部追踪**: [Google MediaPipe Hands](https://google.github.io/mediapipe/solutions/hands.html)
- **核心语言**: JavaScript (ES6 Modules)

## 快速开始

1.  在现代浏览器（如 Chrome, Edge, Firefox）中打开 `index.html` 文件。
2.  当浏览器请求时，请**允许**网站访问您的摄像头。
3.  等待 "正在初始化..." 提示消失。
4.  将您的手放到摄像头前，开始与粒子共舞吧！
